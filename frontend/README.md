# Smart Expense AI Frontend

Frontend application for Smart Expense AI.

## Stack

- React
- TypeScript
- Vite
- Tailwind CSS
- Recharts
- Lucide React

## Getting Started

From the `frontend` directory:

```bash
npm install
npm run dev
```

The application will run by default at:

```txt
http://localhost:5173
```

## Available Scripts

```bash
npm run dev
npm run build
npm run preview
npm run lint
```

## Current Status

The current version contains an initial dashboard prototype with:

- Sidebar navigation.
- Financial summary metrics.
- Monthly spending chart.
- AI alert cards.
- Recent transactions table.
- Mock data prepared for future API integration.
